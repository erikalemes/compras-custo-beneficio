# Código de Conduta

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/), versão 2.1.

## Resumo

- Trate todas as pessoas com respeito.
- Aceite críticas construtivas.
- Foque no que é melhor para a comunidade.
- Não serão tolerados assédio, ataques pessoais ou discriminação.

Violações podem ser reportadas abrindo uma issue com a tag `conduta` ou
contactando as pessoas mantenedoras do repositório.
